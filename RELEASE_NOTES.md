# Release Notes: NotesBar v2.0 - The Spotlight Update

I am excited to announce a major update to NotesBar, bringing deeper system integration and expanded note-taking support.

## 🚀 New Features

### 🍏 Apple Notes Integration
NotesBar now speaks Apple Notes.
- **Unified Access**: Access your Apple Notes alongside your Obsidian vaults.
- **Fast Preview**: Instant Markdown-style previews for your Apple Notes directly in the menu bar app.
- **Direct Opening**: Click any Apple Note to open it instantly in the native macOS Notes app.

### 🔍 Global Spotlight Search (`Ctrl + N`)
Find anything, from anywhere, instantly.
- **System-wide Shortcut**: Press `Ctrl + N` from any app to bring up the new Spotlight-style search bar.
- **Live Previews**: Navigate through search results with arrow keys to see high-fidelity previews without opening the file.
- **Cross-Vault Search**: Search across all your Obsidian vaults and Apple Notes simultaneously.

### 🔦 macOS Spotlight Integration
Your notes are now a core part of your Mac.
- **System Indexing**: Find your notes using the system-wide `Cmd + Space` search.
- **Tab to Search**: Type "NotesBar" in Spotlight and press `Tab` to enter a dedicated search mode for your notes (macOS 15+).
- **Customizable Privacy**: New settings allow you to toggle whether notes appear in general system search or only via "Tab to Search."

### ⚙️ Native Settings Redesign
A completely overhauled settings experience.
- **Native Aesthetic**: Redesigned from the ground up to follow macOS Human Interface Guidelines.
- **Categorized Controls**: Easily manage Keyboard Shortcuts and Spotlight Integration via a familiar Tab-based interface.

## 🛠️ Improvements & Fixes
- **Performance**: Improved file indexing speed using concurrent background processing.
- **Stability**: Fixed a recurring crash during multi-vault synchronization.
- **UI Polishing**: Optimized search list layouts to prevent content overlap and improved keyboard navigation focus.
- **Cleanup**: Removed the **Vault Graph** feature due to its buggy nature and low utility, focusing on core note discovery and previewing.
