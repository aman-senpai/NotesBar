# Release Notes - NotesBar v1.0.0

The 1.0.0 release marks a major milestone for NotesBar, introducing advanced visualization, rich markdown support, and significant performance optimizations for Obsidian users.

## Major Enhancements

### Interactive Vault Visualization
- Integrated a force-directed graph to visualize connections between notes.
- Support for zooming, panning, and clicking nodes to navigate the vault.

### Rich Markdown Previews
- Native support for **Mermaid.js** diagrams, including flowcharts and sequence diagrams.
- **KaTeX** integration for high-quality mathematical and scientific notation.
- Enhanced task list rendering with interactive checkbox support.

### Platform Performance
- Rebuilt the markdown engine on WebKit for smoother rendering and better CSS compatibility.
- Resolved hover-state lag for instantaneous previews in the menu bar.
- Initial search results are now indexed for faster keyword matching across large vaults.

## Bug Fixes and Refinement
- Fixed an issue where nested markdown checkboxes failed to render correctly.
- Improved the floating window management to ensure persistent visibility during multitasking.
- Refined the menu bar controller logic to prevent accidental popup dismissals.

## Installation
The v1.0.0 release is available for download as a ZIP archive on the GitHub Releases page.
